# React Enterprise Toolkit 🚀

A collection of production-ready React patterns focused on solving common enterprise bottlenecks.

## 🛠 Features

- **Azure AD SSO Integration:** Pre-built `AuthGuard` using MSAL for secure, redirect-based authentication.
- **Robust State Management:** Clean Redux Toolkit patterns with standardized Loading/Error handling.
- **Performance Optimized:** Modular components designed for zero-re-render impact.

## 💡 Why this exists?

Enterprise apps fail at the "edges"—authentication redirects and complex global state. These patterns are designed to be "plug-and-play" for teams using Azure and Redux.
